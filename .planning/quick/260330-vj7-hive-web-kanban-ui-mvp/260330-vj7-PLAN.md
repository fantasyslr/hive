**Hooks:**

`hooks/useBoard.ts`:
- State: `{ agents: RegisteredAgent[], tasks: Task[], loading: boolean, error: string | null }`
- On mount: call `fetchBoard()`, populate agents and tasks
- Expose `setAgents` and `setTasks` setters so SSE hook can update state
- Expose `refresh()` to force re-fetch

`hooks/useSSE.ts`:
- Takes `{ onEvent: (event: HiveEvent) => void }` callback
- Creates `EventSource('/api/events/stream/public')`
- State: `connected: boolean`
- Listen to all event types via `evtSource.onmessage` (better-sse sends typed events, so also add listeners for `task.assigned`, `task.updated`, `task.completed`, `task.failed`, `agent.online`, `agent.offline`)
- Parse JSON data, call `onEvent` with reconstructed HiveEvent
- On error: set connected=false, EventSource auto-reconnects
- On open: set connected=true
- Cleanup on unmount: `evtSource.close()`
- Return `{ connected }`
