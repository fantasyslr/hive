<objective>
Fix two high-severity bugs found in Codex adversarial review:

1. **Template watcher crash on missing directory** -- `startTemplateWatcher()` unconditionally calls `fs.watch(dirPath)` which throws ENOENT on clean checkout where `templates/` doesn't exist, crashing the gateway process.

2. **Template launch silently drops unresolved deps** -- `launchTemplate()` uses `.filter()` to silently discard dependencies whose titles don't match any task, instead of rejecting the request. This hides template authoring errors and creates DAGs with missing edges.

Purpose: Prevent gateway crash on fresh deployments; enforce template DAG integrity matching /tasks/batch rollback semantics.
Output: Hardened template-loader.ts and templates.ts with corresponding test coverage.
</objective>
