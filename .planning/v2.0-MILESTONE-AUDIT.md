## Root Cause

All gaps share one root cause: **gateway `index.ts` doesn't wire LlmClient or HistoryInjector into the services that need them**. Each phase executor created correct code in isolation, but the gateway startup file wasn't fully updated to pass all dependencies.

## Fix (3 changes in index.ts + 1 in package.json + 1 in worker-bridge.ts)

1. Import `createHaikuClient` from `@hive/worker` and `HistoryInjector` from `./services/history-injector.js`
2. Construct LlmClient: `const llmClient = process.env.ANTHROPIC_API_KEY ? createHaikuClient(process.env.ANTHROPIC_API_KEY) : null`
3. Pass `llmClient` to MemoryService, CoordinatorService constructors
4. Instantiate HistoryInjector and pass to Dispatcher
5. Add `@hive/worker` to hive-gw dependencies
6. Add startSession/endSession calls in worker-bridge for persistent sessions
